# ipo-lr-14

## Задачи

1. Создать два связанных класса:
    - Класс для базовой работы с изображениями (загрузка, изменение размеров, сохранение).
    - Класс для применения эффектов и обработки изображений (фильтры, добавление текста и других элементов).
2. Реализовать взаимодействие классов для выполнения задач обработки изображений.
3. Разработать методы для выполнения специфических операций с изображениями согласно вариантам задания.
4. Отладить разработанные программы и протестировать их на разных изображениях.
5. Представить результаты работы в виде изменённых изображений и кода, соответствующего варианту.
